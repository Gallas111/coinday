## GAME SDK — AI 에이전트 배포 도구

Virtuals Protocol의 핵심은 GAME SDK(Generative Autonomous Multimodal Entity)예요. 개발자가 AI 에이전트를 Base 체인 위에 배포·운영·수익화하는 도구 세트예요.

**GAME SDK 핵심 기능**

- **모듈식 에이전트 빌더**: LLM(GPT-5.5·Claude 4.7·Gemini 3.1) 연결, 메모리·도구·플러그인 결합
- **온체인 에이전트 ID**: 각 에이전트가 Base 체인 위 NFT 형태로 발행, 소유권·수익권 명확화
- **수익 분배 자동화**: 에이전트가 생성한 수익(API 호출료·구독료·콘텐츠 판매)을 토큰 보유자·개발자·플랫폼에 분배
- **Bonding Curve 토큰 발행**: 각 에이전트가 자신의 sub-토큰을 Bonding Curve 방식으로 발행 가능
- **에이전트 간 상호작용**: 에이전트가 다른 에이전트의 API·도구를 호출하는 멀티 에이전트 협업

이 구조는 ① 개발자가 GAME SDK로 에이전트를 만들고 ② 토큰 발행으로 자금을 모으고 ③ 에이전트 사용 수익이 토큰 보유자에게 분배되는 사이클을 만들어요. 단 모든 에이전트가 실제 수익을 만드는 건 아니고 활성 1,180개 중 월간 활성(MAU 100+) 에이전트는 약 280개 수준이라는 자체 발표가 있어요.

## Bonding Curve — 에이전트 sub-토큰 발행

Virtuals Protocol의 차별점 중 하나는 각 에이전트가 자신의 sub-토큰을 Bonding Curve 방식으로 발행할 수 있다는 점이에요. Bonding Curve는 토큰 발행량과 가격이 수학 공식으로 연결되는 자동화된 발행 메커니즘이에요.

**Bonding Curve 작동 방식**

- **초기 발행**: 에이전트 개발자가 GAME SDK로 sub-토큰 발행 (예: AIXBT, LUNA-BOT 등)
- **가격 공식**: 토큰 가격 = f(공급량). 일반적으로 P = a × supply^k 형태 (k는 곡선 기울기)
- **자동 매수·매도**: 사용자가 매수하면 공급량 증가 → 가격 상승, 매도하면 공급량 감소 → 가격 하락
- **유동성 자동 생성**: 별도 DEX 풀 필요 없이 Bonding Curve가 유동성 역할
- **VIRTUAL 매수**: sub-토큰 매수 시 VIRTUAL 일부 락업 → VIRTUAL 가격에 우호 작용

이 구조는 ① 에이전트 개발자가 즉시 토큰화 가능 ② 사용자가 에이전트별 토큰에 투자 가능 ③ VIRTUAL이 모든 에이전트의 베이스 토큰 역할을 하는 사이클을 만들어요. 단 Bonding Curve는 유동성이 낮은 sub-토큰에서 미끄러짐(slippage)이 크고 가격 변동성이 매우 높아요. 일부 sub-토큰은 24시간 ±50% 이상 변동도 흔해요.

## Base 체인 효과 — Coinbase 생태계

Virtuals Protocol은 Base 체인 위에서 운영돼요. Base는 Coinbase가 출시한 이더리움 L2(OP Stack)로 2024~2026년 빠르게 성장 중인 체인이에요. Base 위 dApp이라는 점이 VIRTUAL에 미치는 효과를 정리했어요.

**Base 체인 효과 정리**

- **Coinbase 사용자 노출**: Coinbase Wallet·Coinbase 거래소 사용자가 Base 체인 dApp에 자연스럽게 노출
- **낮은 가스비**: Base 평균 가스비 약 $0.005~0.02 수준, AI 에이전트 빈번한 트랜잭션에 적합
- **OP 슈퍼체인 인터옵**: Base ↔ OP Mainnet ↔ Mode ↔ Zora 등과 자산 이동 간소화
- **Coinbase Smart Wallet**: 비번 기반 지갑, 신규 사용자 진입 장벽 낮음
- **USDC 네이티브 발행**: Circle이 Base에 USDC를 네이티브로 발행, 결제·정산에 유리

이런 Base 효과는 Virtuals Protocol이 ① 사용자 진입 장벽 낮음 ② 가스비 부담 적음 ③ 자산 이동 간편의 세 가지 이점을 누리는 구조를 만들어요. 단 Base 자체가 Coinbase 단일 시퀀서 운영 구조라 중장기 분산화 진행이 필요하고 Base 의존도가 너무 높으면 Base 리스크가 Virtuals에 직접 전이될 수 있어요.

A: AI 에이전트 섹터에서 가장 강한 시총 성장 속도예요. 활성 에이전트 수 +52% 증가 등 펀더멘털 지표 동반 상승이 있지만 시총 상승 폭이 펀더멘털 대비 크고 단기 모멘텀 영향이 포함돼 있어요. 추세 지속·조정 여부는 단정하기 어려운 구간이에요.

### Q: Bonding Curve로 sub-토큰을 매수하면 안전해요?

A: Bonding Curve는 유동성 자동 생성 구조지만 sub-토큰 유동성이 낮을 때 미끄러짐(slippage)이 크고 매수·매도 가격 격차가 커요. 일부 sub-토큰은 24시간 ±50% 이상 변동이 흔하고 활성 사용자가 없는 에이전트의 sub-토큰은 사실상 가치가 0에 가까울 수 있어요.

### Q: VIRTUAL vs ai16z 어디가 더 큰 프로젝트예요?

A: 시총 기준 VIRTUAL($4.85억) > ai16z($3.2억)이고 활성 에이전트 수도 VIRTUAL이 더 많아요. 단 ai16z는 Solana 기반 Eliza 프레임워크가 오픈소스로 강하고 멀티 LLM 지원 폭이 더 넓다는 차별점이 있어요. 두 프로젝트는 같은 섹터 안에서 다른 접근법을 취하고 있어요.

### Q: AI 에이전트 섹터가 알트코인 시즌의 시작일까요?

A: AI 에이전트 섹터의 강세는 알트코인 시즌 전체 흐름이 아니라 "섹터 시즌"의 한 사례로 해석돼요. 2026년 알트코인 시장은 RWA·AI·Perp DEX·DePIN 같은 섹터별 차별화 흐름이 강하고 모든 알트가 동반 상승하는 2021년식 시즌은 아니에요. 5월 16일 발행한 [JPMorgan 알트 약세 분석](/blog/jpmorgan-altcoin-underperformance-bitcoin-dominance-may-2026)에서 본 약세 흐름과 AI 섹터 강세는 공존하는 구조예요.
